📌 核心理念

KDJ随机振荡指标是短线操作的利器，在指示行情转折点方面具有明显的优势。 本书围绕"KDJ指标"和"波段操作"两个关键词，从指标构成、应用原则到交易系统，系统讲解了KDJ的全部用法。 书中首次引入KD-MACD柱状线概念，将MACD的柱线分析方法移植到KDJ系统中， 并总结了四撞顶(底)等独创形态。

关键词: 精通一个指标 → 万法同理 → 波段为王 → 顺势操作 → 严格执行

1
指标应用基础
6节

核心要点: 技术分析指标分为趋势指标(MA/BOLL/DMI)和振荡指标(MACD/KDJ/RSI)两大类。

  • 区间振荡指标: 数值在0-100区间波动(如KDJ)，>80超买，<20超卖
  • 中心振荡指标: 围绕0轴波动(如MACD)，多空分界线=0轴
  • 多空分界线: KDJ的50线 / MACD的0轴 — 简单但实战意义重大
  • 指标高位与低位: 超买=买过头了(短期看多) → 超卖=卖过头了(短期看空)
  • 指标的趋势: 波峰波谷逐波提高=上升趋势，逐波降低=下降趋势
  • 指标的形态: 头肩顶(底)、双重顶(底)、三重顶(底) — 形态与价格互相验证
  • 指标的交叉: 金叉=快线自下而上穿慢线(买入)，死叉=反之(卖出)
  • 指标的背离: 最好用的应用 — 价格创新高(低)但指标不创新高(低)
2
KDJ指标构成
7节 · 核心

由乔治·兰恩(George Lane)博士于20世纪50年代末发明

📐 计算公式

RSV = (Close - Low_N) / (High_N - Low_N) × 100
K = 2/3 × K_prev + 1/3 × RSV (RSV的3日SMA)
D = 2/3 × D_prev + 1/3 × K (K的3日SMA)
J = 3K - 2D (放大K与D的差距)
KD-MACD = 2×(K-D) ★首创★

📊 四条线的特点

RSV: 未成熟随机值，波动最剧烈
K快线: RSV平滑后，灵敏但有效
D慢线: K的平滑，更可靠但滞后
J线: J>100超买，J<0超卖，最领先

  • RSV(未成熟随机值): 收盘价在N日价格通道中的百分比位置(0-100)
  • K快线: RSV的3日移动平均，灵敏反映收盘价在通道中的相对位置
  • D慢线: K的3日移动平均，D慢线的转向信号通常发出在价格底部/顶部形成之前
  • J线: J=3K-2D，可大于100和小于0，起到领先指标的作用
  • KD-MACD柱线: 2×(K-D)，用MACD柱线方式表示K与D的差距 ★书中首创★
  • 兰恩建议: 配合涨跌周期、波浪理论、斐波那契回撤判断时机
3
K快线波段操作
6节 · 实战核心

K快线是KDJ中最灵敏的操作线，6种波段操作方法:

🟢 买点

第一买点: K快线从超卖区向上突破50线
第二买点: 回落后再次突破50线(通常=主升浪起点)
超卖突破买点: K快线向上突破20线
背离第一买点: 底背离后K快线拐头向上
背离第二买点: 底背离后突破前期高点(更可靠)

🔴 卖点

第一卖点: K快线从超买区向下跌破50线
第二卖点: 反弹后再次跌破50线(=主跌浪开始)
超买跌破卖点: K快线向下跌破80线
背离第一卖点: 顶背离后K快线拐头向下
背离第二卖点: 顶背离后跌破前期低点

  • 趋势线运用: K快线突破下降趋势线=买点(比突破50线更提前)
  • 指标形态: 头肩顶/底、双重顶/底、三重顶/底 → 在超买/超卖区出现更可靠
  • 指标钝化: 极端行情中K快线在超买/超卖区反复振荡，失去指导意义
  • 应对钝化4策: ①扩大周期 ②配合中长线指标 ③等待反向极限位 ④K线形态+量价确认
4
K快线与D慢线波段操作
5节 · 核心

KD线的聚散交叉是KDJ研究的重点:

  • 黄金交叉(金叉):
    • 20线以下金叉: 最强买入信号 — 超卖区的反转
    • 50线以下金叉: 上涨趋势中的加仓点(通常=第3浪起点)
    • 80线以下金叉: 最后一浪上涨(第5浪), 谨慎加仓
    • 80线以上金叉: 延长浪, 宜观望
  • 死亡交叉(死叉):
    • 80线以上死叉: 最强卖出信号 — 超买区的反转
    • 80线以下死叉: 波段高位的短线卖点
    • 50线以下死叉: 下跌趋势中的出逃机会
    • 20线以下死叉: 弱势反弹失败, 继续看空
  • 二次交叉: 同一区域发生两次金叉/死叉 → 第二次更可靠
  • 回测不破: K回调到D线附近但不死叉 → 强势延续(买入)
  • 穿越不过: K反弹到D线附近但不金叉 → 弱势延续(卖出)

KD-MACD柱状线 ★首创★
7节 · 创新

将MACD柱线分析方法引入KDJ, 形成KD-MACD = 2×(K-D):

  • 柱线破解: 红柱(K>D)=多头主导, 绿柱(K
  • 缩头: 红柱从增长转为缩短 → 看跌信号(多头力量减弱)
  • 抽脚: 绿柱从增长转为缩短 → 看涨信号(空头力量减弱)
  • 杀多棒: 红柱突然大幅缩短 → 空头突袭, 多头被杀
  • 逼空棒: 绿柱突然大幅缩短 → 多头反击, 空头被逼
  • 单峰(谷)形态: 柱线只形成一个峰/谷 → 快速反转信号
  • 双重峰(谷)形态: 柱线形成两个峰/谷 → 第二个更可靠
  • 三重峰(谷)形态: 三个峰/谷 → 强势钝化后的反转
  • 柱线背离: 价格创新高/低但柱线峰值降低 → 反转预警

KDJ指标的综合应用
4节
  • 四撞顶(底): ★独创★ KDJ在超买(超卖)区间反复触碰4次以上 → 极强反转信号
    • 四撞顶: K线4次进入>80区间后向下拐头 → 顶部确认
    • 四撞底: K线4次进入<20区间后向上拐头 → 底部确认
  • KDJ顶(底)背离:
    • 80以上的背离可能是假背离(强势延长浪中)
    • 20-50之间的背离最可能是真背离
    • D值跌破50后的背离卖点更可靠
  • 多周期共振: 日线+周线KDJ同时发出信号 → 大幅提高成功率
  • 波浪理论买点: 在第2浪回调结束时的KDJ低位金叉 = 主升浪(第3浪)入场点
7
KDJ与其他指标
6节
  • KDJ与K线形态: KDJ金叉+看涨K线形态(锤头线/吞没) → 双重确认
  • KDJ与均线: KDJ金叉+价格站上MA10 → 买入成功率大增
  • KDJ与成交量: KDJ金叉+放量 → 确认反转; 缩量金叉 → 可能假信号
  • KDJ与MACD: KDJ低位金叉+MACD的DIF>0 → 顺势操作, 高胜率
  • KDJ与布林线: K线触及布林下轨+KDJ超卖 → 强力买入; 触及上轨+KDJ超买 → 强力卖出
  • KDJ神奇变形: 调整参数(如19,3,3或5,3,3)适应不同周期行情
8
市场研判
5节
  • 多空分界: KD值>50=多头市场(做多有利), KD值<50=空头市场(做空有利)
  • 长期vs短期: 周线KDJ判断大趋势, 日线KDJ精确入场出场
  • 多空转折点:
    • K线从超卖区突破50线 = 空转多(第一次最有效)
    • K线从超买区跌破50线 = 多转空(第一次最有效)
  • 市场强弱: 强势市场K快线在50-80游走, 弱势市场在20-50游走
  • 左侧vs右侧交易: KD-MACD提供左侧预警(柱线缩头/抽脚), KD交叉提供右侧确认
9
KDJ交易系统
4节 · 系统化
  • KDJ超买系统:
    • 入场: K快线在超卖区(K<20)向上突破20线
    • 出场: K快线在超买区(K>80)向下跌破80线
    • 适用: 震荡行情, 波段操作
  • KD-MACD系统:
    • 入场: KD-MACD柱线由绿转红(零轴上穿)
    • 出场: KD-MACD柱线由红转绿(零轴下穿)
    • 辅助: 缩头=减仓预警, 抽脚=加仓信号
  • 操盘高手经验:
    • 万法同理 — 精通一个指标, 其他指标极易上手
    • 顺势操作 — 不与趋势作对
    • 概率思维 — 技术分析有效性以概率形式出现
    • 严格执行 — 按信号操作, 不被情绪左右
